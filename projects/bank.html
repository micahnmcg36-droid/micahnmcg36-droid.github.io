About This Project

Based on a competition from Kaggle. Predicted whether customers would subscribe to a bank term deposit. I built this project in both R and Python; see GitHub for both files.

Data Cleaning

I ran some basic procedures to clean and examine both the numerical and categorical variables, and then I split the data into training and testing and ran a standard Random Forest Model to make predictions. Overall both models (R and Python) seemed to perform well, based on the AUC scores. In python I used the sci-kit learn library, while in R I used Caret and some of R's built in functionality. This was a great exercise for me to brush up on R, by doing something I am very comfortable with in Python.
